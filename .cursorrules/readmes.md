## 기술 스택 마이그레이션
- Frontend: Next.js 15.3.0 기반으로 업그레이드
- UI 컴포넌트: 기존 레거시 UI → Tailwind CSS + Headless UI
- 상태 관리: 전역 상태는 React Context로 관리, 서버 상태는 React Query 도입
- 데이터 로딩: Server Components와 Server Actions 활용
- 인증: NextAuth.js로 통합
